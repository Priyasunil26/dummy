This directory contains comprehensive documentation for the Bold Reports Utilities deployment system.

## Building the Documentation

The documentation is built using **MkDocs** with the **Material** theme.

### Prerequisites

Install MkDocs and required dependencies:

```bash
pip install mkdocs mkdocs-material pymdown-extensions
```

### Build Commands

#### Serve Locally (Development)

```bash
mkdocs serve
```

This starts a local web server at `http://127.0.0.1:8000/` with live reloading.

#### Build Static Site

```bash
mkdocs build
```

This generates a static site in the `site/` directory.

#### Deploy to GitHub Pages

```bash
mkdocs gh-deploy
```

This builds and deploys the documentation to GitHub Pages.

## Troubleshooting

### Mermaid diagrams not rendering

Make sure you have the Mermaid JavaScript included in `mkdocs.yml`:

```yaml
extra_javascript:
  - https://unpkg.com/mermaid@10/dist/mermaid.min.js
```

### Build errors

Clear the site directory and rebuild:

```bash
rm -rf site/
mkdocs build
```

### Theme issues

Ensure Material theme is installed:

```bash
pip install --upgrade mkdocs-material
```
