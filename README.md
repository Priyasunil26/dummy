## Quick Start

### Linux Deployment

```bash
# Install Bold Reports
sudo bash build/linux/install-boldreports.sh

# Install optional client libraries
sudo bash build/clientlibrary/Linux/install-optional.libs.sh
```

**Documentation**: [Linux Deployment Scripts](docs/01_linux_deployment_scripts.md)

---

### Docker Single-Container Deployment

```bash
# Build image
docker build -t boldreports-single build/dockerfiles/latest/single-docker-image/

# Run container
docker run -d --name boldreports-single boldreports-single
```

**Documentation**: [Docker Single-Container Deployment](docs/04_docker_single_container_deployment.md)

---

### Docker Multi-Container Deployment

```bash
# Build each service
docker build -t boldreports-api build/dockerfiles/latest/boldreports-server-api.txt

# Run containers
docker run -d --name boldreports-api boldreports-api
```

**Documentation**: [Docker Multi-Container Deployment](docs/05_docker_multi_container_deployment.md)

---

### Kubernetes Deployment

```bash
# Build images
docker build -t boldreports-api:latest -f build/dockerfiles/latest/boldreports-server-api.txt .

# Deploy to K8s
kubectl apply -f k8sfiles/your-deployment.yaml
```

**Documentation**: [Kubernetes Deployment](docs/06_kubernetes_deployment.md)
